# ESX Routen / Geldwäsche Script

Ein kleines FiveM Routen / Geldwäsche Script für ESX.

## Features
- Konfigurierbare Routen mit Sammelpunkt, Verarbeitungspunkt und Verkaufspunkten (zeitbasiert anpassbar)
- Geldwäsche System
- NPC Spawning mit eigenen Models und Animationen
- Blips für alle Standorte
- Discord Webhooks für alle Spieleraktionen
- Animations und Prop Unterstützung
- Automatische Inventarprüfung (stoppt wenn voll)

## Installation
1. Resource in den `resources` Ordner ziehen
2. `ensure bmx-routen` in der `server.cfg` hinzufügen
3. `config.lua` anpassen nach eigenem belieben
4. Server neustarten

## Voraussetzungen
- ESX Framework
- oxmysql / mysql-async
